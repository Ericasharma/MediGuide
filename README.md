## 📌 About the Project  
Health-Assistant is a smart health-support web application that uses Machine Learning to provide insights based on user input.  
It includes preprocessing, model inference, visualization, and a clean UI — ready for deployment on Render, Vercel, or any cloud platform.

---

## ✨ Features  
- 🧠 **ML-powered predictions**  
- 📊 **Interactive health analytics** using Plotly  
- ⚡ **Fast API responses**  
- 🎨 **Responsive UI** with Bootstrap  
- 🔐 Secure & production-ready configuration  
- 🚀 Deployable on Render / Vercel / Heroku

## 🛠️ Getting Started  

### ✔ Prerequisites  
- Python 3.x  
- Pip  
- Virtual environment (recommended)

---

### 📥 Installation  

```bash
# 1. Clone the repository
git clone https://github.com/rajsirohi23/Health-Assistant.git
cd Health-Assistant

# 2. Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate   # Linux/macOS
# or
venv\Scripts\activate      # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure environment (optional)
# Edit config.py to add secret keys, paths, etc.

# 5. Run the application
python app.py
'''bash

```

# 🚀 Deployment Guide

Render
1. Create new Web Service
2. Connect your GitHub repo
3.  Set build command:

```
pip install -r requirements.txt
```

4.  Set start command:
```
gunicorn app:app
```
5. Deploy
